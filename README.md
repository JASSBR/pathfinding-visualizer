# Pathfinding Algorithm Visualizer

Visualize A*, Dijkstra, BFS, DFS on a grid with walls, weights, and speed control.

## Tech Stack

- HTML
- CSS
- JavaScript
- React

## Getting Started

```bash
# Clone the repo
git clone https://github.com/JASSBR/pathfinding-visualizer.git
cd pathfinding-visualizer

# Open in browser
open index.html
```
